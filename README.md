This is a basic yet functional calculator application built using Flutter. The app allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division. It features a clean UI with responsive buttons and a display area that shows both the input and the final calculated result. The project is ideal for beginners learning Flutter layouts, state management, and handling user interactions. It works smoothly on both Android and iOS devices.
